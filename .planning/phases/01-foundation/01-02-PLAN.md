         - If no user: redirect to /login
         - If user but NOT reviewer (user.app_metadata?.reviewer !== true): redirect to /login?error=access_denied
      6. For /login page with authenticated reviewer: redirect to /dashboard
      7. Return supabaseResponse
    - export const config with matcher: ['/((?!_next/static|_next/image|favicon.ico|.*\\.(?:svg|png|jpg|jpeg|gif|webp)$).*)']

    - Redirect non-reviewers to /login?error=access_denied (not just /login)
