## Color

All values map to CSS custom properties defined in `apps/admin/src/app/globals.css`. Use semantic token names in code — never raw hex.

| Role | Light Token | Light Value | Dark Token | Dark Value | Usage |
|------|------------|-------------|-----------|------------|-------|
| Dominant (60%) | `--background` | #F9FAFB | `--background` | #111827 | Page background |
| Secondary (30%) | `--card` | #FFFFFF | `--card` | #1F2937 | TtsPlayer strip background; confirm dialog body |
| Accent (10%) | `--primary` | #F6A5B3 | `--primary` | #F6A5B3 | "生成する" CTA (audio-absent state); confirm dialog confirm button |
| Destructive | `--destructive` | #EF4444 | `--destructive` | #F87171 | Not used in this phase |

Accent reserved for:
1. "生成する" button (audio-absent state — the primary CTA to create the first TTS)
2. Confirm dialog confirm button ("再生成する")
3. Focus ring (`--ring: #F6A5B3`) on play/pause button and field selector

Player strip in normal state (audio present):
- Background: `--card` (#FFFFFF / #1F2937)
- Border: `--border` (#E5E7EB / rgba(255,255,255,0.1))
- Play/Pause button: `variant="outline"` — NOT accent (secondary action; reviewer already has the audio)
- Regenerate button: `variant="ghost"` with `RotateCcw` icon — NOT accent (secondary/optional action)
- Primary visual anchor in audio-present state: Play/Pause button (leftmost action element after field selector).

Audio-absent state:
- Player strip background: `--muted` (#F3F4F6 / #374151)
- "オーディオなし" label: `--muted-foreground` (#6B7280 / #9CA3AF)
- "生成する" button: `variant="default"` — accent (primary CTA, the only action available)

Cooldown state:
- Regenerate button: `disabled` with clock icon — `--muted-foreground` text color
- Countdown label: `--muted-foreground` text, 12px (Label role)

Source: Phase 3 UI-SPEC color contract + `globals.css` — carried forward with Phase 4 player-specific additions.

### TtsPlayer Strip Layout

Single row, full-width, `border border-border rounded-lg bg-card p-3`:

```
[field selector ▾ — w-40]  [gap-2]  [▶/⏸ Play/Pause — 32px icon button]  [waveform bars — flex-1]  [↺ — ghost icon button]
```

- Field selector: `w-40` fixed, `h-8` compact Select
- Play/Pause button: `variant="outline" size="icon"` — 32px, `Play` or `Pause` from lucide
- Waveform: 3–4 animated `div` bars (`bg-primary/60`, height animates with `animate-pulse` only when `isPlaying`). When not playing: static bars at half-height. When no audio: hidden.
- Regenerate button: `variant="ghost" size="icon"` with `RotateCcw` lucide icon. When cooldown active: `Clock` icon + countdown text label beside it (replaces RotateCcw area).

### Audio-Absent State (D-02)

When no TtsAudio record exists for the item:

```
┌─────────────────────────────────────────────────────┐
│ [field selector ▾]   [オーディオなし]   [+ 生成する]   │
│ bg-muted, no play button, no waveform               │
└─────────────────────────────────────────────────────┘
```

- Background: `bg-muted`
- "オーディオなし" label: `text-muted-foreground text-sm`
- "生成する" button: `variant="default"` (accent/primary) — triggers RegenerateConfirmDialog directly

## Interaction States

### TtsPlayer — Audio present, not playing
| State | Visual |
|-------|--------|
| Default | `variant="outline"` Play button; static waveform bars; `variant="ghost"` RotateCcw button |
| Field selector change | Immediately fetches TTS for new field; play button disabled during fetch |

### TtsPlayer — Playing
| State | Visual |
|-------|--------|
| Playing | `variant="outline"` Pause button (swaps icon); waveform bars animate (`animate-pulse`) |
| Audio ends naturally | Returns to Play button state; waveform static |

### TtsPlayer — Audio absent (D-02)
| State | Visual |
|-------|--------|
| Default | `bg-muted` strip; "オーディオなし" muted label; accent "生成する" button |
| "生成する" click | Opens RegenerateConfirmDialog immediately |

### Regenerate button (D-03, D-04)
| State | Visual |
|-------|--------|
| Default | `variant="ghost" size="icon"` RotateCcw icon |
| Clicked | Opens RegenerateConfirmDialog |
| In cooldown (D-07) | Button shows `Clock` icon + "N分後" countdown label beside it; `disabled`; `text-muted-foreground` |

### RegenerateConfirmDialog (D-03, D-04)
- Title: "{itemLabel}のTTSを再生成しますか？" (20px semibold)
- Body: none — keep dialog minimal per D-03
- Confirm button: "再生成する" — `variant="default"` (accent). When mutation pending: `disabled` + Loader2 `animate-spin`
- Cancel button: "キャンセル" — `variant="outline"`
- On confirm: mutation fires; dialog closes on success
- On success (D-04, D-05): success toast fires; new audio URL loaded into player; auto-play starts; cooldown timer starts
- On error: error toast fires; dialog closes; regenerate button returns to default state

### Cooldown countdown (D-06, D-07)
| State | Visual |
|-------|--------|
| Cooldown active | Regenerate area replaced with: `Clock` icon (16px, muted) + "N分後に再生成可能" label (12px, muted-foreground) |
| Countdown ticks | Label updates every second via `setInterval` |
| Cooldown expires | Cooldown label disappears; RotateCcw button re-appears |

### Loading skeleton (TtsPlayer initial fetch)
| State | Visual |
|-------|--------|
| Query loading | Player strip shows: field selector skeleton (`animate-pulse bg-muted rounded w-40 h-8`) + play button skeleton (32px circle) + waveform skeleton (3 bars) |

## Accessibility Notes

- Play/Pause button: `aria-label="再生" / "一時停止"` toggled with play state. Screen reader announces state change.
- Regenerate button: `aria-label="TTSを再生成"`. When disabled during cooldown: `aria-disabled="true"` with `aria-describedby` pointing to countdown label.
- RegenerateConfirmDialog: Radix DialogPrimitive — focus trap, `aria-modal="true"`, `role="dialog"`. Focus lands on cancel button first (safer default for irreversible action).
- Field selector: shadcn Select uses Radix SelectPrimitive — keyboard navigable, `aria-expanded`, `aria-selected` managed automatically.
- Waveform animation: `aria-hidden="true"` — decorative only; not conveying state to screen readers.
- Cooldown countdown: `role="status"` so screen reader announces updates without focus movement.
