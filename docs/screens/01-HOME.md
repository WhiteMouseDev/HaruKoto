## 섹션 상세

### 1. 헤더 영역

| 항목 | 설명 |
|------|------|
| 위치 | 최상단 |
| 좌측 | 시간대별 인사말 (좋은 아침/오후/저녁) + 사용자 닉네임 |
| 우측 | 알림 벨 아이콘 (NotificationCenter) + 읽지않은 알림 뱃지 |

### 2. PhoneCallCta (AI 음성통화 배너)

| 항목 | 설명 |
|------|------|
| 스타일 | 보라색 그라데이션 카드 |
| 내용 | "AI 통화" 레이블 + Beta 뱃지 + 통화 아이콘 |
| 동작 | 클릭 → `/chat/call/contacts` (연락처 목록) |

### 3. KanaCtaCard (가나 학습 유도)

| 항목 | 설명 |
|------|------|
| 조건 | 사용자 JLPT 레벨이 N5이고 가나(히라가나/가타카나) 학습 미완료 시에만 표시 |
| 내용 | 진행 중인 가나 타입에 따라 "히라가나 배우기" 또는 "가타카나 배우기" 표시 |
| 동작 | 클릭 → `/study/kana` |

### 4. StreakBadge + DailyProgressCard (가로 병렬)

| 항목 | 설명 |
|------|------|
| StreakBadge | 🔥 불꽃 아이콘 + 연속 학습 일수 + 요일별 체크 표시 (월~일) |
| DailyProgressCard | 3개 지표 그리드: 목표 진행(n/목표), 학습 단어수, 정답률(%) |
| 애니메이션 | 숫자 카운트업 애니메이션 |

### 5. DailyMissionsCard (오늘의 미션)

| 항목 | 설명 |
|------|------|
| 헤더 | "오늘의 미션" + 완료 카운트 (n/전체) |
| 미션 목록 | 아이콘 + 미션명 + XP 보상 + 완료 체크 |
| 보상 버튼 | 완료된 미션의 XP 보상 수령 (Claim) |
| 프로그레스 | 전체 미션 진행률 바 |

### 6. QuickStartCard (학습 시작)

| 항목 | 설명 |
|------|------|
| 스타일 | 그라데이션 배경 카드 |
| 내용 | 사용자 JLPT 레벨 기반 학습 시작 CTA |
| 동작 | 클릭 → `/study` |

### 7. WeeklyChart (주간 학습 차트)

| 항목 | 설명 |
|------|------|
| 차트 | 막대 차트 (월~일, 요일별 학습 단어수) |
| 기준선 | 점선으로 일일 목표 표시 |
| 하단 | 이번 주 총 학습량 표시 |

### 8. LevelProgress (JLPT 레벨 진도)

| 항목 | 설명 |
|------|------|
| 표시 | N5 → N4 → N3 → N2 → N1 수평 진행 |
| 상태 | 완료(●), 현재(진행중), 잠김(○) |
| 활성 | 현재 N5, N4만 활성화 |

## 데이터 의존성

| API 엔드포인트 | 용도 |
|----------------|------|
| `GET /api/v1/stats/dashboard` | 대시보드 전체 데이터 (스트릭, 진행률, 주간차트) |
| `GET /api/v1/missions/today` | 오늘의 미션 목록 + 완료 상태 |
| `GET /api/v1/user/profile` | 사용자 닉네임, JLPT 레벨, 가나 진행도 |

## 애니메이션

- 전체 섹션: Framer Motion `staggerChildren`으로 순차 fade-in
- 숫자: 카운트업 애니메이션 (DailyProgressCard)
- 카드: hover 시 scale 효과
