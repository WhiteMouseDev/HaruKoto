## 결제 플로우 전체

```
/pricing → 플랜 선택 → /subscription/checkout → PortOne 결제
                                                     │
                                            ┌────────┴────────┐
                                            │                 │
                                       결제 성공          결제 실패
                                            │                 │
                                  /subscription/success   에러 표시
                                            │            (재시도 안내)
                                    구독 상태 활성화
```

## 데이터 의존성

| API 엔드포인트 | 용도 |
|----------------|------|
| `GET /api/v1/subscription` | 현재 구독 상태 조회 |
| `POST /api/v1/payments/create` | 결제 요청 생성 |
| `POST /api/v1/webhook/portone` | PortOne 결제 결과 웹훅 수신 |
| `POST /api/v1/subscription/activate` | 구독 활성화 |
