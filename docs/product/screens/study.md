| 나의 학습 데이터 | 오답노트, 학습한 단어, 단어장, 50음도 차트 바로가기 |

```
┌─────────────────────────────────┐
│  ← 뒤로   N5 단어 퀴즈   7/20  │  ← 헤더 (뒤로가기, 제목, 진행수)
│  ████████████░░░░░░░░░░░░░░░░  │  ← 프로그레스 바
├─────────────────────────────────┤

│         「 食べる 」            │  ← 문제 (일본어 대형 텍스트)
│          たべる                 │  ← 서브 텍스트 (읽기/발음)
│                                 │
│                                 │
│  ┌──────────────────────┐      │
│  │  ① 먹다              │      │  ← 4지선다 옵션
│  └──────────────────────┘      │    - 기본: 회색 테두리
│  ┌──────────────────────┐      │    - 정답 선택: 초록 배경 + pulse
│  │  ② 마시다            │      │    - 오답 선택: 빨강 배경 + shake
│  └──────────────────────┘      │    - 미선택: opacity 감소
│  ┌──────────────────────┐      │
│  │  ③ 보다              │      │
│  └──────────────────────┘      │
│  ┌──────────────────────┐      │
│  │  ④ 듣다              │      │
│  └──────────────────────┘      │
│                                 │
│              [💡 힌트]          │  ← 힌트 버튼 (전구 아이콘)
│                                 │
│  ┌─────────────────────────────┐│
│  │ ✅ 정답! "먹다"             ││  ← 정답/오답 피드백 패널
│  │ 예문: 朝ごはんを食べる      ││    (답 선택 후 나타남)
│  │      아침밥을 먹다          ││    오답 시 [📕 단어장 저장] 버튼
│  │              [다음 →]       ││
│  └─────────────────────────────┘│
│                                 │
└─────────────────────────────────┘
```

### 인터랙션 흐름

1. 문제 표시 → 사용자 4지선다 중 택 1
2. 선택 즉시 정답/오답 피드백 (색상 + 애니메이션)
3. 피드백 패널 노출 (정답 해설, 예문)
4. 오답 시 "단어장 저장" 버튼 표시
5. "다음" 클릭 → 다음 문제
6. 마지막 문제 → 결과 화면 이동
