# 마이페이지 (My) — `/my`

> 사용자 프로필, 업적, 구독 상태, 앱 설정을 관리하는 화면.

## 파일 위치

- **페이지**: `apps/web/src/app/(app)/my/page.tsx`
- **주요 컴포넌트**:
  - `components/features/my/profile-hero.tsx` (page.tsx 내부 inline)
  - `components/features/my/achievements-section.tsx` (page.tsx 내부 inline)
  - `components/features/my/subscription-section.tsx`
  - `components/features/my/settings-menu.tsx` (page.tsx 내부 inline)

## 데이터 의존성

| API 엔드포인트 | 용도 |
|----------------|------|
| `GET /api/v1/user/profile` | 프로필 정보 (닉네임, 아바타, 레벨, XP, 통계) |
| `PATCH /api/v1/user/profile` | 닉네임 수정, JLPT 레벨 변경, 목표 변경 |
| `GET /api/v1/subscription` | 구독 상태 조회 |
| `GET /api/v1/payments/history` | 결제 내역 목록 |
