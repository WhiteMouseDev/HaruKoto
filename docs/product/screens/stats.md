# 통계 (Stats) — `/stats`

> 사용자의 학습 데이터를 기간별, 학습별, JLPT 진도별로 시각화하는 화면.

## 파일 위치

- **페이지**: `apps/web/src/app/(app)/stats/page.tsx`
- **주요 컴포넌트**:
  - `components/features/stats/heatmap.tsx`
  - `components/features/stats/bar-chart.tsx`
  - `components/features/stats/period-tab.tsx`
  - `components/features/stats/study-tab.tsx`
  - `components/features/stats/jlpt-tab.tsx`

## 섹션 상세

| 탭 | 주요 요소 | 설명 |
|-----|----------|------|
| 기간별 | Heatmap | GitHub 스타일 연간 학습 히트맵. 연도 전환 가능. 색상 강도로 학습량 표현. hover 시 툴팁 |
| 기간별 | BarChart | 주간/월간/연간 토글. 막대 차트로 학습 시간 시각화 |
| 기간별 | 오늘 요약 | 오늘의 학습시간 + 퀴즈 횟수 요약 |
| 학습별 | 카테고리 카드 | 단어/문법/한자/회화 4개 카테고리별 정답률, 진행률, 학습량 |
| JLPT진도 | 현재 레벨 | 현재 JLPT 레벨의 단어/문법 진행률 상세 |
| JLPT진도 | 레벨 현황 | N5~N1 전체 레벨 상태 (완료/현재/잠김) |
