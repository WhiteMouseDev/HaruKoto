### 섹션 상세

| 섹션 | 설명 |
|------|------|
| 서브 탭 | "음성통화" / "텍스트 회화" 전환 |
| PhoneCallCta | 음성통화 탭에만 표시. 보라색 그라데이션 배너 |
| CategoryGrid | 2×2 카테고리 격자: 여행, 일상, 비즈니스, 자유대화 |
| ScenarioCard | 카테고리 선택 후 시나리오 목록. 난이도/시간/핵심표현 표시 |
| ConversationHistory | 과거 대화 기록. 점수 뱃지, 삭제, 페이지네이션 |

### 인터랙션 흐름

1. 시나리오 시작 → 상황 설명 카드 표시
2. AI 첫 메시지 (일본어)
3. 사용자 텍스트 입력 → 전송
4. AI 응답 (TypingIndicator → 메시지)
5. 반복...
6. "대화 종료" → 피드백 화면으로 이동

## 데이터 의존성

| API 엔드포인트 | 용도 |
|----------------|------|
| `POST /api/v1/chat/start` | 대화 세션 시작 (시나리오 + 캐릭터) |
| `POST /api/v1/chat/end` | 대화 종료 + 피드백 생성 요청 |
| Vercel AI SDK | 스트리밍 AI 응답 (텍스트 채팅) |
| WebRTC / 음성 API | 실시간 음성 통화 |
