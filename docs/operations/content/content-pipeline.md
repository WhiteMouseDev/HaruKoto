## 1. 역할 분담

### PM(당신)이 하는 것

```
① 레슨 블루프린트 작성 (레슨당 5분)
   - 제목, 주제, 대화 상황 한 줄
   - 단어 5~8개 선정 (기존 N5 데이터에서 order 번호로)
   - 문법 1개 배정 (기존 문법 order 번호로)

② N5 문법 추가 (현재 30개 → 80개)
   - JLPT N5 표준 문법 목록 기반
   - AI가 초안 → 당신이 검수
   - 패턴 + 뜻 + 예문 2개

③ 단어 주제 태깅 검수
   - AI가 606개 단어에 카테고리 태그 초벌
   - 당신이 확인/수정

④ 최종 검수 (레슨당 20분)
   - 대화문 자연스러운지
   - 문제 적절한지
   - 오답이 정답과 겹치지 않는지
```

### AI(Claude)가 하는 것

```
① 블루프린트 → 레슨 콘텐츠 자동 생성
   - 대화문 (화자별 voice_id 포함)
   - 확인 문제 5개 (인식2 + 적용2 + 산출유사1)
   - 오답 선택지 (유사도 기반)
   - 정답 설명

② N5 문법 초안 생성
   - JLPT 표준 목록에서 패턴 + 뜻 + 예문

③ 단어 태깅 초벌
   - 606개 단어에 카테고리 자동 배정

④ 자동 QA
   - N5 범위 밖 단어 사용 체크
   - 오답/정답 뜻 중복 체크
   - 문제 커버리지 체크

⑤ TTS 생성
   - ElevenLabs로 화자별 음성 생성
```

---

## 2. 제작 순서

```
Step 1: PM이 블루프린트 스프레드시트 작성
Step 2: AI가 레슨 콘텐츠 생성 (대화문 + 문제)
Step 3: 자동 QA (스키마/난이도/커버리지)
Step 4: PM 검수 (JP 자연스러움 + KO 정확도)
Step 5: 확정 → DB 적재
Step 6: TTS 생성
```

## 3. 블루프린트 템플릿

### 스프레드시트 형식

| 필드 | 설명 | 예시 |
|------|------|------|
| lesson_id | 레슨 고유 ID | N5-P1-C1-L1 |
| title | 레슨 제목 | 인사하기 |
| topic | 주제 | 인사 |
| scene | 대화 상황 (한 줄) | 회사 첫 출근, 아침 인사 |
| roles | 대화 화자 | 신입사원, 선배 |
| vocab_orders | 단어 order 번호들 | 196, 362, 364, 372, 161 |
| grammar_order | 문법 order 번호 | 1 |
| can_do | 학습 목표 (한 줄) | 시간대별 인사를 하고 응답할 수 있다 |
| notes | 특이사항 | - |

### JSON 형식 (AI 입력용)

```json
{
  "lesson_id": "N5-P1-C1-L1",
  "title": "인사하기",
  "topic": "인사",

  "notes": ""
}
```

### PM 검수 체크리스트

```
□ 대화문이 자연스러운 일본어인가?
□ 지정된 단어 5개가 모두 등장하는가?
□ 지정된 문법(〜です)이 사용되었는가?
□ 문제 5개가 인식2+적용2+산출유사1 구성인가?
□ 오답이 정답과 뜻이 겹치지 않는가?
□ N5 범위 밖 단어가 사용되지 않았는가?
□ 한국어 번역/설명이 정확한가?
```
