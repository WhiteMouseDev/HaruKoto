## 진행 순서

의견에 따라 작업 범위가 유연하게 달라질 수 있습니다.

### Step 1. 서비스 분석 + 의견 수렴

앱을 써본 후 아래 항목에 대해 솔직한 피드백을 부탁드립니다.

| 항목            | 현재 상태                                            | 듣고 싶은 의견                                 |
| --------------- | ---------------------------------------------------- | ---------------------------------------------- |
| **서비스명**    | "하루코토(ハルコト)" — 개발자가 즉흥적으로 지은 이름 | 괜찮은지, 바꿔야 한다면 어떤 방향이 좋을지     |
| **로고**        | 벚꽃 심볼 + 워드마크 — 개발자가 직접 만든 것         | 다듬는 수준으로 충분한지, 새로 만들어야 하는지 |
| **컬러**        | 체리핑크 + 봄 컨셉 — 감으로 잡은 색상                | 이 방향이 맞는지, 완전히 다른 톤이 나을지      |
| **마스코트**    | 여우 캐릭터 "하루" — 거의 활용되지 않는 상태         | 살려서 적극 활용할지, 빼는 게 나을지           |
| **전반적인 UI** | 전부 개발자가 만든 디자인                            | 가장 시급하게 손봐야 할 곳은 어디인지          |

> 현재 서비스명, 로고, 컬러, 마스코트 등 **확정된 것이 하나도 없습니다.**
> 전부 개발 과정에서 임시로 만든 것이므로, 전문가 관점에서 자유롭게 판단해 주세요.

### Step 2. 브랜드 방향 확정

Step 1 의견에 따라 범위가 결정됩니다.

- 서비스명 변경 → 네이밍 + 로고 신규 제작
- 서비스명 유지 → 로고 정제 또는 리디자인
- 로고 큰 문제 없음 → 폰트 통일 + 색상 정리 수준

### Step 3. 앱 아이콘 / 피처 그래픽 / 스플래시 스크린

로고가 확정되면 파생 에셋을 제작합니다.

| 작업                    | 규격                   | 비고                      |
| ----------------------- | ---------------------- | ------------------------- |
| 앱 아이콘               | 1024x1024 PNG 원본 1장 | iOS/Android/PWA 전부 사용 |
| Google Play 피처 그래픽 | 1024x500 PNG 1장       | 스토어 상단 배너          |
| 스플래시 스크린         | 로고 중앙 배치         | 현재는 빈 흰 화면         |

### Step 4. 앱 UI 개선

핵심 화면 시안 작업입니다. (Step 1 의견에 따라 우선순위 및 범위 조정)

| 화면                                              | 우선순위 |
| ------------------------------------------------- | -------- |
| 홈 / 학습 탭 / 퀴즈 / 채팅 (AI 회화) / 마이페이지 | 높음     |
| 로그인 / 온보딩 / 가나 학습                       | 보통     |

### Step 5. 랜딩 페이지

서비스 소개 + 앱 다운로드 유도 페이지입니다. (1페이지)

## 현재 상태 (자체 평가)

디자인 전부 개발자가 직접 작업한 것으로, 전문 디자이너의 손을 거치지 않았습니다.

- 핑크 색상값이 3~4개 혼용 중 (`#F59BA7`, `#F6A5B3`, `#FFB7C5`) → 통일 필요
- 로고 폰트가 한글(산돌 아그로체)과 일본어(을지로체)로 달라서 통일감 부족
- 여우 마스코트가 있지만 앱 내에서 거의 사용되지 않음
- 앱 아이콘은 벚꽃 심볼 + "ハル" 텍스트 구성
- 스플래시 스크린은 빈 흰색 화면

## 개발 환경 (참고)

디자인 시안을 실제 코드로 구현하는 환경입니다.

| 항목          | 기술                                             |
| ------------- | ------------------------------------------------ |
