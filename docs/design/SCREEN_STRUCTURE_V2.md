## 1. 전체 화면 플로우

### 1.1 비로그인 (최초 설치)

```
스플래시 → 앱 소개 슬라이드 → 로그인/회원가입 → 온보딩 → 홈
```

- **스플래시**: 로고 표시 (1초 이내)
- **앱 소개 슬라이드**: 서비스 핵심 가치 전달 (2~3장)
- **로그인**: 카카오 + 구글 소셜 로그인
- **온보딩** (현행 유지):
  1. 닉네임 입력
  2. JLPT 레벨 선택 (N5~N1)
  3. 가나 학습 여부 (N5 선택 시에만)
  4. 학습 목표 선택 (JLPT 급수별 / 여행 / 비즈니스 / 취미)
- 온보딩 완료 후 showKana=true → 홈 (가나 학습 유도)

### 1.2 로그인 유저 (재방문)

```
스플래시 → 로딩 (데이터 프리페치) → 홈
```

- **로딩**: 대시보드 데이터, 유저 프로필, 학습 진도 등 사전 로드
- 데이터 로드 완료 시 즉시 홈으로 전환 (최소 체감 로딩)

## 3. 홈 화면 (`/home`)

### 3.1 구조

```
┌──────────────────────────┐
│  헤더 (인사 + 알림)       │
├──────────────────────────┤
│  연속 학습 (스트릭)       │  ← 컴팩트, 탭하면 달력 바텀시트
├──────────────────────────┤
│                          │
│  오늘의 학습 카드         │  ← 메인 영역
│  ┌────────────────────┐  │
│  │ 탭: 단어 | 문법 | 문장 │  │
│  │                    │  │
│  │ 추천 학습 콘텐츠    │  │
│  │ (복습할 단어 12개)  │  │
│  │                    │  │
│  │ [학습 시작] CTA     │  │
│  └────────────────────┘  │
├──────────────────────────┤
│  학습 정보               │  ← 탭하면 학습통계 탭으로 이동
├──────────────────────────┤
│  바로가기 (4개)           │
│  [단어장] [오답노트]      │
│  [도전과제] [가나차트]    │
└──────────────────────────┘
```

### 3.2 섹션별 상세

#### 연속 학습 (스트릭) 섹션
- **높이 최소화** — 말해보카 스타일로 컴팩트하게
- 연속 일수 + 간단한 아이콘
- **탭 → 달력 바텀시트** (월별 학습 기록 표시)

#### 오늘의 학습 카드
- 탭으로 카테고리 전환: **단어 | 문법 | 문장**
- 현재 추천 탭의 로직 활용 (복습할 단어, 새로운 단어, 오답노트)
- 눈에 띄는 **[학습 시작] CTA 버튼**

#### 학습 정보
- 오늘 푼 문제 수, 정답률 등 간단한 요약
- 탭하면 학습통계 탭으로 이동

#### 바로가기 (4개)
- 단어장, 오답노트, 도전과제, 가나 차트(50음도)
- 2x2 그리드 배치

## 4. 학습 페이지 (`/study`)

### 4.1 구조 변경 (Before → After)

**Before (현행)**:
```
추천/자율 탭
  └── 자율 → N5/N4/N3/N2/N1 레벨 선택
       └── 단어/문법 카테고리 선택
            └── 4지선다/매칭/빈칸/어순/쓰기 모드 선택
                 → 퀴즈
```
문제: 4단계 depth, 선택 피로

**After (개선)**:
```
단어 | 문법 | 문장배열 | [가나]  ← 단일 탭 레벨
  └── 스테이지 리스트 (현재 JLPT 레벨 기준)
       └── 스테이지 탭 → 모드 선택 바텀시트
            → 풀스크린 퀴즈
```
개선: 2단계 depth, JLPT 레벨은 설정에서 변경

### 4.2 탭 구성

```
┌──────────────────────────────────┐
│  [단어]  [문법]  [문장배열]  [가나] │  ← 동적 가나 탭
├──────────────────────────────────┤
│  현재 레벨: N5  (변경 →)         │  ← 설정으로 이동
├──────────────────────────────────┤
│                                  │
│  스테이지 1: N5 동사 기초         │
│  ████████░░ 80%                  │
│  4지선다 ✅  매칭 ✅  빈칸 ○      │
│                                  │
│  스테이지 2: N5 형용사 기초       │
│  ██░░░░░░░░ 20%                  │
│  4지선다 ○  매칭 ○  빈칸 ○       │
│                                  │
│  스테이지 3: N5 명사 기초         │
│  🔒 (스테이지 2 완료 후 해금)     │
│                                  │
└──────────────────────────────────┘
```

### 4.3 동적 가나 탭

| 유저 상태 | 탭 구성 | 가나 위치 |
|----------|---------|----------|
| 초보자 (N5 + showKana=true + 가나 미완료) | **가나** \| 단어 \| 문법 \| 문장배열 | 첫 번째 |
| 가나 완료 or showKana=false | 단어 \| 문법 \| 문장배열 | 제거됨 |
| 설정에서 가나 다시 켬 | 단어 \| 문법 \| 문장배열 \| **가나** | 마지막 (복습용) |

- 가나 마스터 시 축하 다이얼로그: "가나를 마스터했어요! 탭을 숨길까요?"
- 유저 확인 후 탭 제거 (설정에서 복원 가능)

### 4.4 JLPT 레벨

- 학습 페이지 상단에 현재 레벨 표시 (예: "현재 레벨: N5")
- 탭하면 설정의 레벨 변경으로 이동
- **레벨 선택 탭(N1~N5)은 제거** — 유저가 자주 바꾸는 항목이 아님

### 4.5 스테이지 → 퀴즈 모드 선택

스테이지를 탭하면 **모드 선택 바텀시트** 표시:

```
┌─────────────────────────┐
│  N5 동사 1               │
│                         │
│  모드: 📝 4지선다        │
│                         │
│  하루 목표: 10문제  ✏️   │
│                         │
│  [학습 시작]             │
└─────────────────────────┘
```

- **하루 목표**는 마이페이지가 아닌 **학습 진입 시점에서 설정** (말해보카 방식)
- 카테고리별로 다른 목표 설정 가능
- 연필 아이콘(✏️) 탭하면 문제 수 변경 (5/10/15/20/30)

**카테고리별 사용 가능 모드**:

| 카테고리 | 사용 모드 | 비고 |
|---------|----------|------|
| 단어 | 4지선다, 매칭, 빈칸 | 의미 인식 → 짝짓기 → 문맥 활용 |
| 문법 | 4지선다, 빈칸 | 규칙 인식 → 적용 |
| 문장배열 | 어순 배열 (전용) | 모드 선택 없이 바로 시작 |
| 가나 | 기존 스테이지 학습 | 별도 플로우 (intro→연습→퀴즈→리뷰) |

- **쓰기(타이핑) 모드**: 한국인 유저의 일본어 키보드 사용 현실을 고려하여 당분간 비활성
- 문장배열/가나는 모드 선택 없이 바로 진입

### 4.6 스테이지 카드 — 모드별 진도 표시

```
┌──────────────────────────┐
│  N5 동사 1               │
│  ████████░░ 80%          │
│  4지선다 ✅  매칭 ✅  빈칸 ○ │
└──────────────────────────┘
```

- 전체 진도(%) + 모드별 완료 여부
- 유저가 자연스럽게 여러 모드를 시도하도록 유도 (강제 아닌 유도)

## 5. 퀴즈 화면 (풀스크린)

### 5.1 공통 규칙

- **탭바 숨김** — 퀴즈 중 하단 네비게이션 제거
- **상단**: 프로그레스 바 + 닫기(X) 버튼
- **닫기 시**: "퀴즈를 종료할까요?" 확인 다이얼로그
- **완료 시**: 퀴즈 결과 화면 → 홈 또는 다음 스테이지

### 5.2 퀴즈 유형별 화면

| 유형 | 화면 구성 | 카테고리 |
|------|----------|---------|
| 4지선다 | 문제 + 4개 선택지 | 단어, 문법 |
| 매칭 | 좌측 일본어 ↔ 우측 한국어 짝짓기 | 단어 |
| 빈칸 채우기 | 문장 + {빈칸} + 선택지 | 단어, 문법 |
| 어순 배열 | 한국어 번역 + 토큰 재배열 | 문장배열 |
| 가나 인식 | 가나 문자 + 4지선다 | 가나 |

## 8. 실전회화 페이지 (`/chat`)

```
┌──────────────────────────┐
│  실전회화                  │
```

### 8.2 난이도 설정

JLPT 레벨과 연동하여 자동 설정 (수동 변경 가능):

| 회화 난이도 | JLPT 대응 | 설명 |
|-----------|----------|------|
| 초급 | N5~N4 | 인사, 자기소개, 간단한 주문 |
| 중급 | N3~N2 | 일상 대화, 의견 표현, 상황 설명 |
| 고급 | N1 | 토론, 비즈니스, 뉘앙스 표현 |

- 유저의 JLPT 레벨에 맞춰 기본 난이도 자동 설정
- 시나리오별 난이도 표시

### 8.3 통화 설정 (마이페이지에서 이동)

- 침묵 시간 설정 (0~5초)
- 자막 표시 토글
- 자동 분석 토글

| 항목 | Before (현행) | After (v2) |

| 쓰기(타이핑) 모드 | 사용 가능 | **비활성** (일본어 키보드 이슈) |
| 홈 추천 학습 | 학습 탭 내 추천 탭 | **홈 메인 카드로 이동** |
| 퀴즈 중 탭바 | 표시됨 | **숨김 (풀스크린)** |
| 홈 바로가기 | 없음 | **단어장, 오답노트, 도전과제, 가나차트** |
| 스트릭 표시 | 일반 카드 | **컴팩트 + 탭→달력 바텀시트** |
| 하루 목표 설정 | 마이페이지 학습 설정 | **학습 페이지 진입 시 바텀시트** (카테고리별 개별 설정) |
| 업적 | 마이페이지에 표시 | **홈 바로가기 "도전과제"로 이동** |
| 통화 설정 | 마이페이지 앱 설정 | **실전회화 페이지 내로 이동** |
| 학습통계 탭 | 오늘/주간/월간 | **기간별/학습별/JLPT진도** 3탭 |

## 11. 디자이너 참고사항

1. **최소 탭으로 핵심 액션 도달** — 홈에서 2탭 이내로 퀴즈 시작
