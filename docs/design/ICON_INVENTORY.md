## 디자이너 참고사항

1. **아이콘 라이브러리**: [Lucide Icons](https://lucide.dev/icons/) — 전체 아이콘 목록은 공식 사이트에서 확인
2. **기본 크기**: 16px(small), 20px(medium), 24px(large), 32px(extra-large)
3. **색상**: 브랜드 핑크, 시맨틱 색상(success/error/warning), 또는 theme.onSurface with alpha
4. **Apple 로고**: Material Icons 유지 (Lucide에 브랜드 아이콘 없음)
5. **커스텀 아이콘 필요 시**: `assets/icons/` 디렉토리에 SVG로 추가
