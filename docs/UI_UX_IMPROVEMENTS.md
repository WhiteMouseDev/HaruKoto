## P0 — 버그/크리티컬

### 1. 랜딩 페이지 CTA 라우팅 오류
- **파일**: `app/page.tsx`
- **문제**: "시작하기" 버튼과 "로그인" 버튼 모두 `/home`으로 이동
- **수정**: "시작하기" → `/login` (회원가입 모드), "로그인" → `/login`

### 2. 비밀번호 찾기 없음
- **파일**: `app/(auth)/login/page.tsx`
- **문제**: 비밀번호 분실 시 복구 방법 없음
- **수정**: Supabase `resetPasswordForEmail` 활용, "비밀번호 찾기" 링크 추가

### 3. 학습 진행률 하드코딩 "0%"
- **파일**: `app/(app)/study/page.tsx:178`
- **문제**: 학습 진행률이 항상 `0%`로 표시 (API 미연동)
- **수정**: 사용자의 실제 학습 진행률을 API에서 fetch

### 4. 단어/문법 수 하드코딩
- **파일**: `app/(app)/study/page.tsx:168-169`
- **문제**: `'200개 단어'`, `'30개 문법'` 하드코딩
- **수정**: DB에서 해당 레벨의 실제 단어/문법 수를 조회

### 5. 통계 페이지 메트릭 불일치
- **파일**: `app/(app)/my/page.tsx`
- **문제**: "총 학습일수"에 `totalQuizzesCompleted` 값 사용
- **수정**: 실제 학습한 고유 일수 계산

## P1 — UX 개선 (사용성)

### 6. href="#" 플레이스홀더 제거
- **파일**: `app/(app)/study/page.tsx`
- **문제**: "내가 학습한 단어" 링크가 `href: '#'` (작동 안 함)
- **수정**: 페이지 구현 또는 "준비 중" 표시 + 비활성화

### 7. 단어장 삭제 확인 없음
- **파일**: `app/(app)/study/wordbook/page.tsx`
- **문제**: 단어 삭제 시 확인 다이얼로그 없이 바로 삭제
- **수정**: 확인 다이얼로그 추가 또는 되돌리기(undo) toast 구현

### 8. 온보딩 JLPT 레벨 선택 혼란
- **파일**: `app/(auth)/onboarding/page.tsx`
- **문제**: N4~N1이 보이지만 선택 불가 → 앱이 고장난 것처럼 보임
- **수정**: 비활성 레벨에 "준비 중" 뱃지 추가, 또는 N5만 표시

### 9. Premium 뱃지 but 무료 접근
- **파일**: `app/(app)/chat/page.tsx`, `components/layout/bottom-nav.tsx`
- **문제**: AI 회화에 "Premium" 뱃지가 있지만 실제 과금 없음
- **수정**: 뱃지 제거 또는 "Beta" 뱃지로 변경 (추후 구독 도입 시 복원)

### 10. 퀴즈 결과에서 레벨업 진행도 미표시
- **파일**: `app/(app)/study/result/page.tsx`
- **문제**: XP 획득은 보이지만 다음 레벨까지 얼마나 남았는지 표시 없음
- **수정**: XP 프로그레스 바 추가 (현재 XP / 다음 레벨 XP)

### 11. 대화 피드백 점수 해석 없음
- **파일**: `components/features/chat/feedback-scores.tsx`
- **문제**: "유창성 73%" 같은 숫자만 있고 이게 좋은 건지 나쁜 건지 모름
- **수정**: 점수 구간별 간단한 해석 텍스트 추가 (예: "좋아요!", "더 연습해봐요")

### 12. 에러 메시지 일관성
- **파일**: 전체
- **문제**: "데이터를 불러올 수 없습니다" 같은 제네릭 메시지
- **수정**: 상황별 구체적 에러 메시지 + 해결 방법 안내

## P2 — 하드코딩 상수화

### 13. 게임 보상 상수 통합
현재 XP 보상이 여러 파일에 흩어져 하드코딩됨:

| 파일 | 값 | 용도 |
|------|-----|------|
| `api/v1/quiz/complete/route.ts` | `correctCount * 10` | 퀴즈 정답 XP |
| `api/v1/quiz/start/route.ts` | `correctCount * 10` | 미완료 세션 XP |
| `api/v1/chat/end/route.ts` | `20` | 회화 완료 XP |

**수정**: `lib/constants.ts`에 `REWARDS` 상수 생성

### 14. 퀴즈 설정 상수화
| 파일 | 값 | 용도 |
|------|-----|------|
| `study/page.tsx` | `10` | 기본 문제 수 |
| `result/page.tsx` | `80`, `50` | 결과 이모지 분기 기준 |
| `api/v1/quiz/start/route.ts` | `0.6` | 복습:신규 비율 |
| `api/v1/quiz/start/route.ts` | `slice(0, 3)` | 오답 보기 수 |

**수정**: `lib/constants.ts`에 `QUIZ_CONFIG` 상수 생성

### 15. 업적 임계값 상수화
- **파일**: `lib/gamification.ts`
- **문제**: 15개 이상의 업적 임계값이 함수 본문에 하드코딩
- **수정**: `ACHIEVEMENTS` 배열의 각 항목에 `threshold` 필드 추가하여 데이터 기반으로 전환

### 16. 간격 반복(SRS) 매직 넘버
- **파일**: `lib/spaced-repetition.ts`
- **문제**: SM-2 알고리즘 파라미터 하드코딩 (`3초`, `8초`, `interval 1/3/21일`)
- **수정**: `SRS_CONFIG` 상수로 추출

### 17. 페이지네이션 기본값
- **파일**: `api/v1/wordbook/route.ts`, `study/wordbook/page.tsx`
- **문제**: 페이지 크기 `20` 하드코딩
- **수정**: `PAGINATION_DEFAULTS` 상수화

## P3 — 향후 개선

### 18. 프로필 편집 기능
- 온보딩 후 닉네임 변경 불가
- 프로필 사진 업로드 불가

### 19. 계정 삭제
- 계정 탈퇴 기능 없음 (향후 GDPR 대응 필요)

### 20. 접근성(a11y) 보완
- 아이콘 전용 버튼에 `aria-label` 누락
- 색상만으로 상태 구분하는 곳에 텍스트 보조 필요
- 폼 에러 메시지와 입력 필드 연결 (`aria-describedby`)

### 21. 대화 중 메시지 복사
- 일본어 텍스트 길게 눌러 복사하는 기능 없음

### 22. 피드백 표현 → 단어장 추가
- 회화 피드백의 추천 표현을 바로 단어장에 추가하는 기능
