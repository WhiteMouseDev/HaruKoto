## 1. 디자인 시스템

### 1.1 컬러 팔레트 (CSS 변수로 관리)

> **중요**: 모든 컬러는 Tailwind CSS 변수(`--color-*`)로 관리하여
> 브랜드 컬러 변경 시 한 곳만 수정하면 전체 반영되도록 합니다.
> **next-themes**로 라이트/다크 테마를 전환합니다.

```css
/* globals.css - 라이트 테마 (기본: 봄 테마) */
:root {
  --color-primary: #ffb7c5; /* Cherry Pink - 메인 액션 */
  --color-primary-hover: #ff8fa3; /* Deep Pink - 호버/활성 */
  --color-primary-light: #ffe4ec; /* Light Pink - 배경 강조 */

  --color-secondary: #87ceeb; /* Sky Blue - 보조 액션 */
  --color-accent-green: #98d8c8; /* Mint Green - 성공/정답 */
  --color-accent-red: #ff6b6b; /* Coral - 경고/오답 */
  --color-accent-yellow: #ffd93d; /* Warm Yellow - 보상/포인트 */

  --color-bg: #fff8f0; /* Cream White - 메인 배경 */
  --color-bg-card: #ffffff; /* Pure White - 카드 배경 */
  --color-bg-secondary: #fff0f5; /* Soft Pink - 보조 배경 */

  --color-text: #2d2d2d; /* 본문 */
  --color-text-secondary: #6b7280; /* 보조 텍스트 */
  --color-text-muted: #d1d5db; /* 비활성/플레이스홀더 */

  --color-success: #10b981;
  --color-error: #ef4444;
  --color-warning: #f59e0b;
  --color-info: #3b82f6;
}

/* 다크 테마 */
.dark {
  --color-primary: #ff8fa3;
  --color-primary-hover: #ffb7c5;
  --color-primary-light: #3d1f2a;

  --color-secondary: #5ba3c9;
  --color-accent-green: #6bc4ad;
  --color-accent-red: #e05252;
  --color-accent-yellow: #e5c235;

  --color-bg: #1a1a2e; /* 다크 네이비 */
  --color-bg-card: #242442; /* 카드 배경 */
  --color-bg-secondary: #2a2a4a; /* 보조 배경 */

  --color-text: #f0f0f0;
  --color-text-secondary: #9ca3af;
  --color-text-muted: #4b5563;

  --color-success: #34d399;

}
```

### 1.2 타이포그래피

```
[한국어/일본어 본문]
Font: Pretendard (한국어) + Noto Sans JP (일본어)

[일본어 학습 강조]
Font: Noto Serif JP (한자/히라가나 강조 표시용)

[크기 체계]
Display:   32px / Bold    ← 페이지 타이틀
Heading:   24px / Bold    ← 섹션 타이틀
Title:     20px / Semi    ← 카드 타이틀
Body:      16px / Regular ← 본문
Caption:   14px / Regular ← 보조 텍스트
Small:     12px / Regular ← 뱃지, 태그
```

### 1.3 컴포넌트 스타일

```
[카드]
Background: #FFFFFF
Border Radius: 16px
Shadow: 0 2px 8px rgba(0,0,0,0.06)
Padding: 16px~20px

[버튼 - Primary]
Background: #FFB7C5 → hover: #FF8FA3
Text: #FFFFFF
Border Radius: 12px
Height: 48px (모바일)

[버튼 - Secondary]
Background: transparent
Border: 1.5px solid #FFB7C5
Text: #FFB7C5
Border Radius: 12px

[입력 필드]
Background: #F9FAFB
Border: 1.5px solid #E5E7EB → focus: #FFB7C5
Border Radius: 12px
Height: 48px

[뱃지/태그]
Border Radius: 999px (pill)
Padding: 4px 12px
Font: 12px Semi
```

### 1.4 마스코트 캐릭터 (컨셉)

```
이름: 하루 (ハル)
동물: 벚꽃 여우 (きつね)
특징:
- 벚꽃잎이 달린 귀
- 부드러운 핑크/크림 색상
- 다양한 감정 표현 (기쁨, 응원, 슬픔, 놀람 등)
- 학습 상황별로 다른 모습
  - 정답: 기쁘게 뛰는 모습
  - 오답: 위로하는 모습
  - 스트릭: 응원하는 모습
  - AI 회화: 함께 대화하는 모습
```

## 2. 하단 탭 네비게이션

```
┌──────────────────────────────────────────────┐
│  🏠 홈    📊 학습통계   📚 학습    💬 회화    👤 MY  │
└──────────────────────────────────────────────┘

[아이콘 스타일: 라인 아이콘, 활성 시 필드 + Cherry Pink]
```

| 탭           | 아이콘 | 기능                          | 접근         |
| ------------ | ------ | ----------------------------- | ------------ |
| **홈**       | 🏠     | 대시보드, 오늘의 학습, 스트릭 | 무료         |
| **학습통계** | 📊     | 기간별/학습별 통계, 히트맵    | 무료         |
| **학습**     | 📚     | JLPT 퀴즈 (단어/문법/한자)    | 무료         |
| **회화**     | 💬     | AI 상황극, 자유대화           | **프리미엄** |
| **MY**       | 👤     | 프로필, 설정, 구독관리        | 무료         |

## 4. 인터랙션 & 애니메이션 (Framer Motion)

### 4.1 페이지 전환

```
- 탭 전환: 좌우 슬라이드 (200ms)
- 페이지 진입: 아래→위 페이드인 (300ms)
- 모달/시트: 아래→위 슬라이드 (250ms)
```

### 4.2 퀴즈 인터랙션

```
- 선택지 탭: scale(0.95) → scale(1) 바운스 (150ms)
- 정답: 초록색 하이라이트 + 체크 아이콘 + confetti
- 오답: 빨간색 흔들림(shake) + 정답 표시
- 카드 넘기기: 좌우 스와이프 (플래시카드 모드)
- 진행바: 부드러운 width 트랜지션
```

### 4.3 게이미피케이션

```
- XP 획득: 숫자 카운트업 애니메이션
- 레벨업: 풀스크린 축하 애니메이션 + 마스코트
- 스트릭: 불꽃 아이콘 펄스 애니메이션
- 뱃지 획득: 스케일업 + 반짝임
```

### 4.4 AI 회화

```
- AI 메시지: 타이핑 인디케이터 (점 3개 애니메이션)
- 메시지 등장: 아래→위 페이드인 (stagger)
- 번역 토글: 접기/펼치기 (height 트랜지션)
- 피드백 하이라이트: 밑줄 + 팝오버
```
