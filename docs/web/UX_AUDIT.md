# UX 감사 보고서 — 하루코토 v0.1.0

> 2026-03-06 기준. 학습 경험 강화를 위한 냉정한 현황 분석.

---

## 한줄 요약

**애니메이션과 구조는 잘 잡혀있지만, 학습 앱의 핵심인 "소리"가 완전히 빠져있고, 성취감을 주는 피드백이 약하다.**

---

## 1. 치명적 부재 — 소리가 없다

앱 전체에 **오디오가 단 하나도 없습니다.** 이것이 가장 큰 문제입니다.

| 있어야 하는 곳 | 현재 상태 |
|---|---|
| 정답 시 효과음 | ❌ 없음 |
| 오답 시 효과음 | ❌ 없음 |
| 퀴즈 완료 축하 사운드 | ❌ 없음 |
| 단어 TTS 발음 | ❌ 없음 (API는 존재하지만 퀴즈/단어장에 미연결) |
| 가나 문자 발음 | ❌ 없음 (Volume2 아이콘만 있고 onClick 핸들러 없음) |

Duolingo, WaniKani, Anki 모두 소리에 크게 의존합니다. **언어 학습 앱에서 소리 없는 건 음악 앱에서 재생 버튼이 없는 것과 같습니다.**

### 권장 조치 (최우선)
- 정답/오답 효과음 (짧은 chime/buzz, 2개 파일이면 충분)
- 단어 TTS 재생 버튼 (퀴즈 문제, 결과 화면, 단어장)
- 가나 문자 발음 재생 (KanaCharacterCard에 onClick 연결)

---

## 2. 퀴즈 경험 — 잘 만들었지만 도파민이 부족

### 잘 된 것 ✅
- 정답 시 pulse 애니메이션 (`scale: [1, 1.04, 1]`)
- 오답 시 shake 애니메이션 (`x: [0, -8, 8, -6, 6, -3, 3, 0]`)
- 바텀 시트 피드백 패널 (스프링 물리, Duolingo 스타일)
- 옵션 stagger 진입 애니메이션
- 히트 영역 충분 (`whileTap: { scale: 0.98 }`)
- 답변 후 자동 단어장 추가 CTA

### 부족한 것 ⚠️

| 항목 | 현재 | 개선안 |
|---|---|---|
| **퀴즈 내 스트릭** | 없음 | "3연속 정답! 🔥" 토스트 또는 카운터 |
| **XP 획득 피드백** | 완료 후에만 표시 | 정답마다 "+10 XP" 플로팅 텍스트 |
| **타이머** | 내부 추적만, UI 미노출 | 선택적 타이머 바 표시 |
| **콤보/배율** | 없음 | 연속 정답 시 XP 배율 (×1.5, ×2) |
| **진행 바** | 단순 width 바 | 문제별 점(dot) 표시 (Duolingo 스타일) |

### 퀴즈 모드별 세부 이슈

**매칭 모드**
- 오른쪽 컬럼은 오답 시 shake 없음 — 비대칭 피드백
- 라운드 전환 시 전환 애니메이션 없음 (즉시 remount)
- 매칭 완료 시 축하 효과 없음

**빈칸(Cloze) 모드**
- 빈칸에 답이 들어갈 때 "팝" 느낌 부족
- 번역이 답변 전에 이미 노출 → 힌트/스포일러
- 옵션 칩이 작아 모바일에서 탭하기 어려움

**어순배열 모드**
- 드래그앤드롭 없음 (탭만 가능)
- 부분 정답 인정 없음
- 토큰 색상 코딩은 시각적으로 가장 풍부 — 좋음

**타이핑 모드**
- 실제 키보드 입력 불가 (문자 뱅크 선택만)
- 오답 시 정답 표시가 `text-sm`으로 너무 작음
- 오답 해설이 다른 모드보다 빈약

## 3. 결과 화면 — 기본은 되어 있으나 임팩트 부족

### 잘 된 것 ✅
- 정확도 원형 프로그레스 SVG 애니메이션
- 숫자 카운트업 (정확도%, 정답수, XP)
- 80%+ 시 컨페티 파티클
- 오답 아코디언 + 단어장 일괄 저장

### 부족한 것 ⚠️
- 컨페티가 화면 상단에서 비처럼 내려옴 — 중심에서 터지는 burst가 아님. 반클라이맥스
- 축하 효과음 없음
- 스트릭 카운트 미표시
- 결과 아이콘이 정적 (입장 애니메이션만 있고, 루프/표현 애니메이션 없음)
- 레벨업 순간을 놓치기 쉬움 (XP 바가 작은 stat box 안에 있음)
- 소셜 공유 버튼 없음

## 4. 학습 탭 — 거의 정적

학습 탭(`/study`)은 앱에서 **가장 애니메이션이 부족한 페이지**입니다.

| 요소 | 애니메이션 |
|---|---|
| 추천/자율 탭 전환 | CSS transition만 (슬라이딩 인디케이터 없음) |
| 추천 카드 | CSS hover만 |
| 레벨 선택 버튼 | CSS transition만 |
| 모드 선택 버튼 | CSS transition만 |
| 진행률 바 | CSS transition만 |
| 로딩 상태 | **없음** (스켈레톤 없이 데이터 팝인) |

홈 탭의 풍부한 stagger 애니메이션과 대비됩니다. 학습 탭은 유저가 가장 자주 방문하는 페이지인데 가장 밋밋합니다.

### 권장 조치
- 카드 진입 stagger 애니메이션 추가
- 탭 전환 시 슬라이딩 인디케이터 (홈 바텀 네비처럼 `layoutId`)
- 로딩 스켈레톤 추가

## 5. 온보딩 — 첫인상이 가장 약하다

**스텝 간 전환 애니메이션이 없습니다.** 조건부 렌더링으로 콘텐츠가 즉시 교체됩니다.

온보딩은 앱의 **첫인상**입니다. 여기서 "이 앱 잘 만들었다"는 느낌을 줘야 하는데, 현재는 정적 폼 위저드입니다.

### 세부 이슈
- `AnimatePresence` 미사용 — 스텝 전환 시 슬라이드/페이드 없음
- 뒤로/다음 버튼에 화살표 텍스트(`←`, `→`) 사용 — 아이콘과 불일치
- "시작하기 🌸" 버튼에만 이모지 사용 — 앱 전체의 lucide 아이콘과 불일치
- N3~N1 설명에 "콘텐츠 준비 중" 텍스트 — 첫인상에서 미완성 느낌

### 권장 조치
- `AnimatePresence` + 스텝 전환 애니메이션 (좌→우 슬라이드)
- FoxMascot에 미세한 루프 애니메이션 (호흡, 깜빡임)

## 6. 가나 학습 — 앱에서 가장 완성도 높은 플로우

5단계 학습 시퀀스 (소개→플래시카드→매칭→퀴즈→완료)가 잘 설계되어 있습니다.

### 잘 된 것 ✅
- 3D 카드 플립 (`rotateY: 0 → 180`)
- 스와이프 제스처 (`drag="x"`, 80px 임계값)
- 자동 진행 퀴즈 (1초 후 자동 다음 문제)
- 단계 잠금/진행/완료 시각 상태

### 부족한 것 ⚠️
- 마스터 퀴즈 통과 시 컨페티 없음
- 플래시카드 스와이프 시 색상 오버레이 없음 (방향 힌트 부재)
- TTS 발음 버튼 비작동 (아이콘만 존재)
- 획순 애니메이션 없음 (향후 기능)

---

## 7. 디자인 일관성 이슈

### 스켈레톤 애니메이션 불일치
- 홈 탭: `animate-shimmer` (커스텀 그래디언트 스윕)
- 마이 탭: `animate-pulse` (Tailwind 기본)
- 같은 앱에서 두 가지 로딩 패턴은 비전문적으로 보임

### 다크 모드 이슈
- `animate-shimmer`가 white highlight 하드코딩 → 다크 모드에서 너무 밝음
- `PhoneCallCta`가 디자인 토큰 대신 하드코딩 hex 색상 사용
- `themeColor`가 라이트/다크 모두 `#F6A5B3` (핑크) → 다크 모드 브라우저 크롬이 안 어울림

### 기타
- `PhoneCallCta`의 보라색 그래디언트가 앱의 체리핑크 primary와 불일치
- 바텀 네비 `prefetch={false}` → 탭 전환 시 체감 지연
- 대화 피드백 "같은 시나리오 다시하기" 버튼이 실제로는 채팅 목록(`/chat`)으로 이동 — 라벨과 동작 불일치

## 9. 우선순위별 개선 로드맵

### P0 — 즉시 (학습 경험에 직접적 영향)
1. **효과음 추가** — 정답/오답 chime + 퀴즈 완료 효과음 (3개 오디오 파일)
2. **TTS 연결** — 퀴즈 문제에 단어 발음 재생 버튼, 가나 카드 발음
3. **DailyProgressCard 버그 수정** — 목표 통계 값 오류

### P1 — 1~2주 (성취감 강화)
4. **퀴즈 내 스트릭 카운터** — "3연속 정답! 🔥"
5. **정답 시 XP 플로팅 텍스트** — "+10 XP" 화면에 뜨고 사라짐
6. **컨페티 개선** — 상단 비 → 중심 burst
7. **온보딩 스텝 전환 애니메이션** — AnimatePresence 슬라이드
8. **학습 탭 진입 애니메이션** — stagger + 스켈레톤

### P2 — 2~4주 (완성도)
9. 스켈레톤 애니메이션 통일 (`animate-shimmer` 또는 `animate-pulse` 중 하나로)
10. 다크 모드 shimmer/themeColor 수정
11. "같은 시나리오 다시하기" 버튼 동작 수정
12. 바텀 네비 `prefetch` 활성화 (인접 탭)
13. PhoneCallCta 색상을 디자인 토큰으로 교체

### P3 — 향후 (차별화)
14. 매칭 완료 시 라운드 전환 애니메이션
15. 어순배열 드래그앤드롭
16. 단어장 복습 모드 (단어장에서 바로 퀴즈)
17. 가나 획순 애니메이션
18. 소셜 공유 (결과 화면 카드 이미지 생성)
