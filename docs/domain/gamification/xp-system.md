## 1. 레벨 공식

```
level = floor(sqrt(totalXP / 100)) + 1
```

역산: 레벨 L에 도달하는 데 필요한 총 XP = `(L - 1)² × 100`

**구현**: `apps/web/src/lib/gamification.ts` → `calculateLevel()`

## 3. XP 획득 경로

| 활동 | XP | API 라우트 |
|------|------|------------|
| 퀴즈 정답 (1문항) | 10 | `/api/v1/quiz/complete` |
| 텍스트 회화 완료 | 20 | `/api/v1/chat/end` |
| 음성 회화 완료 | 20 | `/api/v1/chat/end` |
| 히라가나/가타카나 퀴즈 | 10 | `/api/v1/kana/quiz` |
| 미션 보상 | 가변 (10~30) | `/api/v1/missions/claim` |

모든 XP 획득 경로에서 `checkAndGrantAchievements()`가 호출되어 업적 및 캐릭터 해금이 자동 체크됨.

## 4. 캐릭터 해금 임계값

| 캐릭터 | 필요 레벨 | 필요 XP | 예상 도달 시간 |
|--------|----------|---------|--------------|
| 소라, 하루 | 1 (기본) | 0 | 즉시 |
| 유키, 카이토 | 3 | 400 | 약 2~3일 활성 사용 |
| 미오, 렌 | 5 | 1,600 | 약 1~2주 |
| 아오이, 리쿠 | 8 | 4,900 | 약 3~5주 |

해금 시 `UserCharacterUnlock` 레코드가 생성되고, 알림 이벤트가 반환됨.
